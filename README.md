# Daflix - uproszczony serwis steramingowy

Tworzę aplikację webową Daflix, wzorowaną na Netflix. Strona pozwala oglądać jeden film, z możliwością pomijania czołówki i sterowania odtwarzaczem. 

## Funkcjonalności 
- Odtwarzanie jednego filmu
- Możliwość pomijania czołówki 
- Sterowanie odtwarzaniem (pauza, odtwarzanie, przewijanie)
- Przechowywanie danych o użytkowniku i filmie w bazie danych 

## Cel projektu 
Celem jest pokazanie działania aplikacji streamingowej w praktyce oraz obsługi odtwarzacza w przeglądarce, z prostą integracją	bazy danych.
